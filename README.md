## Installation Instructions

### Method 1: Copy the Folder (Easiest)
1. **Copy the entire `NigerianMealPlanner` folder** to any location on your computer:
   - Recommended locations: Documents, Desktop, or any folder you prefer
   
2. **Open the app**:
   - Navigate to the folder
   - Double-click `index.html`
   - The app will open in your default web browser

3. **Create a shortcut** (Optional):
   - Right-click `index.html`
   - Select "Create shortcut"
   - Move the shortcut to your Desktop or taskbar

### Method 2: Transfer to Another Computer
1. **Copy all 3 files** to a USB drive, cloud storage, or email them:
   - `index.html`
   - `style.css`
   - `script.js`
   
2. **On the new computer**:
   - Create a new folder (e.g., `NigerianMealPlanner`)
   - Copy all 3 files into this folder
   - Double-click `index.html` to open

### Method 3: Using the ZIP File
1. **Locate** `NigerianMealPlanner.zip` (if provided)
2. **Extract** the ZIP file to your desired location
3. **Open** the extracted folder and double-click `index.html`

## Troubleshooting

### The app won't open
- Make sure all 3 files (`index.html`, `style.css`, `script.js`) are in the same folder
- Try right-clicking `index.html` ‚Üí "Open with" ‚Üí Choose your web browser

### Styles look broken
- Verify that `style.css` is in the same folder as `index.html`
- Clear your browser cache and refresh the page

### Features don't work
- Ensure `script.js` is in the same folder as `index.html`
- Check if JavaScript is enabled in your browser settings
